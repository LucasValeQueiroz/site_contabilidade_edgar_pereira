# Contabilidade Edgar Pereira - Premium Website

Recriação do site da Contabilidade Edgar Pereira com design moderno, responsivo e premium.

## Tecnologias Utilizadas

- **HTML5** semântico
- **CSS3** vanilla com variáveis CSS, CSS Grid, Flexbox e animações
- **JavaScript** vanilla para interações DOM (menu responsivo, animação de números, partículas)
- **Google Fonts**: Inter (Corpo) e Playfair Display (Títulos)
- **Font Awesome 6** para ícones

## Estrutura do Projeto

- `index.html` - Página principal
- `empresa.html` - História e valores
- `servicos.html` - Detalhamento de serviços
- `noticias.html` - Feed de notícias (estático para front-end)
- `contato.html` - Formulário e informações de contato
- `css/style.css` - Design System completo
- `js/main.js` - Lógica de interações no frontend

## Deploy no Vercel

O site está configurado com um `vercel.json` para facilitar o deploy direto no Vercel:
1. Suba este repositório para o GitHub
2. Conecte o repositório na plataforma Vercel
3. O deploy será feito automaticamente, de forma estática e otimizada (cleanUrls e caching headers já configurados).
